How to Create and use your own measures


With Power BI Desktop, you can create your own measures with the Data Analysis Expressions (DAX) formula language, any field that appears with a sigma icon  is numeric, and its values can be aggregated, Sum is the default aggregation for a numeric datatype, but you can easily apply different aggregations like average or count. 


Note:  Actual measure for NumOfMembers below:

NumOfMembers = DISTINCTCOUNT('Fact_HealthCare'[MemberKey])
